# Darts League

A simple web app to track our darts league standings and match history.

## Players
- Will
- Cal
- Tom
- Alex

## Scoring
- **Match format:** Best of 5 legs (first to 3)
- **Points:** 3 points for a win
- **Bonus:** 1 point for highest checkout over 50 in a match

## Data Structure

### players.json
Player roster with id and display name.

### matches.json
Match records including:
- Date
- Both player IDs
- Legs won by each player
- High checkout for each player
- High visit for each player

## Development
Static site - just open `index.html` in a browser.
